# CH4 Cloud Project

## 📌 프로젝트 개요
Spring Boot 기반 AWS 클라우드 배포 프로젝트입니다.  
EC2, RDS, S3, ALB, GitHub Actions를 활용하여 CI/CD 및 파일 업로드 기능을 구현했습니다.
